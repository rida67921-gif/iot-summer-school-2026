This folder should contain circuit diagrams for all week3 projects.

### Recommended Files:

1. **smart_street_light_circuit.png**
   - Hand-drawn or Tinkercad schematic
   - Shows LDR (analog pin A0), PIR (digital pin 3), LED with PWM (pin 9)
   - Connection diagram to Arduino

2. **multi_sensor_circuit.png**
   - DHT11 (pin 2), LDR (A0), HC-SR04 (pins 8, 9)

3. **keypad_lcd_circuit.png**

### How to Add:

- Draw your circuit on paper and photograph it, or
- Use [Tinkercad](https://www.tinkercad.com/), [Fritzing](https://fritzing.org/), or similar tool
- Save as PNG/JPG and upload to this folder
- Reference in project README files

**Note**: These diagrams are graded for clarity and accuracy. Include all components, wiring, and pin labels.
